SuperMines
=
SuperMines is a powerful, easy-to-use, and free-to-use mine plugin for Paper servers.

## Features
* MiniPlaceholders/PlaceholdersAPI support
* Folia support
* ItemsAdder/Nexo support
* Rank system
* Coming soon...
